ESCAPE Help Guide

ESCAPE (Effect Size Calculator for Practical Effects) helps you translate correlations into practical, interpretable results for decisions and communication.

Quick Start

  1. Start analysis from the landing page by selecting Try with Sample Data or uploading your own file. Supported formats: .csv, .xlsx, .sav, and .sas7bdat.
  2. In the sidebar, choose your Predictor (X) and Criterion (Y) variables.
  3. Set Number of bins, adjust Criterion cutoff (Y), and use Predictor percentile (X) to define comparison groups.
  4. Review the outputs across each analysis tab.
  5. Export your results using Export Report and per-chart Download buttons.

Controls Explained

  • Data: Upload a new dataset at any time from the sidebar. The data card shows row and column counts and whether you are using sample or uploaded data.
  • Predictor (X) and Criterion (Y): ESCAPE analyzes the relationship between these two numeric variables.
  • Number of bins: Controls how Predictor (X) is grouped in the Expectancy chart and table.
  • Criterion cutoff (Y): Defines the success threshold for expectancy and BESD outputs. Use Set to mean to auto-fill the criterion mean.
  • Predictor percentile (X): Splits cases into Above vs Below predictor groups. This directly affects Cohen's d, Hedges' g, CLES, BESD, and Icon Array results.

Tab-by-Tab Guide

Summary

  • Shows top-line metrics: correlation (r), Cohen's d, and CLES.
  • Includes a scatterplot with trend line and a plain-language Key Insight panel.

Data & Descriptives

  • Displays your raw data table.
  • Shows descriptive statistics and variance explained from r2.
  • Provides distribution histograms for Predictor and Criterion.

Expectancy

  • Plots the proportion above the criterion cutoff within each Predictor bin.
  • Includes an expectancy table with bin ranges, proportions, frequencies, and sample size.

Effect Sizes

  • Reports Pearson's r, Hedges' g, Cohen's d, d confidence interval, and CLES probability.
  • Shows group means/SDs for Above vs Below predictor groups.
  • Includes overlap density and two BESD views: empirical (cutoff-based) and theoretical (derived from r).

Icon Array

  • Converts CLES into an intuitive visual count of successes vs failures.
  • Customize icon type, total icons, and layout to match your communication context.

Converter

  • Converts published summary effects when raw data are unavailable.
  • Supports input of Pearson's r or Cohen's d, then computes the corresponding translated metrics.

How to Interpret Key Metrics

  • Pearson's r: Direction and strength of the linear relationship between X and Y.
  • r2: Percent of Criterion variance explained by Predictor.
  • Cohen's d: Standardized mean difference between Above and Below predictor groups.
  • Hedges' g: A bias-corrected standardized mean difference.
  • CLES: Probability that a random person from the Above group outperforms one from the Below group.
  • BESD (empirical): 2x2 probabilities computed from your selected predictor/criterion cutoffs.
  • BESD (theoretical): 2x2 probabilities transformed from correlation via 0.5 +/- r/2.

Example interpretation: if CLES is 0.69, then in 69 out of 100 random Above-vs-Below pairs, the Above-group person is expected to score higher on the criterion.

Troubleshooting

  • No results or blank plots: Confirm both selected variables are numeric and contain non-missing values.
  • CLES or BESD unavailable: Your current percentile split may leave too few observations in one group. Try a less extreme predictor percentile.
  • Upload error: Ensure file extension is supported and data are readable.
  • Unexpected values: Verify the current Criterion cutoff (Y) and predictor percentile (X) settings.

Exports

  • Export Report: Generates an HTML report with key metrics, tables, and core plots.
  • Download buttons: Export individual figures (PNG) from major visual panels.

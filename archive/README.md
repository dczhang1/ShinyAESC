# Archive

This directory contains obsolete, deprecated, and unused files from the ESCAPE project.

## Directory Structure

### `versions/` - Deprecated App Versions
- `appv2.r` — Original single-file app (2018)
- `app_v2.R` — Second iteration
- `app_v3.R` — Third iteration

These are not maintained; use the main app (`app.R`) instead.

### `obsolete/` - Obsolete Files
- `assets/` — Duplicate assets directory (use `www/assets/` instead)
- `logo.png` — Duplicate of `www/assets/logo.png`

### `deployment/` - Deployment Artifacts
- `expectancyApp/` — rsconnect deployment files (ShinyApps.io artifacts)

### `docs/` - Archived Documentation
- `Zhang2018_Frontiers.pdf` — Reference paper from the original publication

## Notes

- The current application is `app.R` in the project root
- Active data files are in `data/`
- Active web assets are in `www/`
- Active utility scripts are in `R/`
- Active documentation is in `docs/`
